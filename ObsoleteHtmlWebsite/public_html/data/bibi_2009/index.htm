There are two files for each of the four locations. The "Raw Data" File contains the individual macro invertebrate counts for each of three replicate (samples) from the site. The "Scoring" file contains  scoring for each of three replicates and a composite (average) score. Scores are determined with reference to 10 indicators which are assigned points based upon the array of critters found in the sample. Pointwise, only a 1, 3, or 5 are assigned to each indicator.

This data is being incorporated into the Puget Sound benthic data found at the Puget Sound Streams Benthos website.
