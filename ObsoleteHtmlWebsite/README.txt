
For doing updates to streamkeepers website. 
synchBack pulls lfpstreamkeepers.org website public_html to local copy. 
publish pushes changes back. 

Run in a cygwin bash shell with sopenssh installed. 

usage is to synch with website, make changes then publish

gotchas 
- if you make changes then synch again you lose changes.
